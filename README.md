# NutriPal-Backend

FastAPI backend for NutriPal

## Setup

We use python poetry for dependency management. Once you install python-poetry just run 

```sh
poetry install
```

To view environment info to add to vscode use

```sh
poetry env info
```

## Running

### Local

```sh
poetry shell
python -m app
```

### Production

```sh
./start.sh
```
